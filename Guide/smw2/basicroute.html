Basic Route

Before you begin focusing on levels, you must first understand the basics of speedrunning SMW2.

When speedrunning SMW2, you shouldn't select difficulty because that ends up putting you in a comic that wastes a bit of time. Plus, resetting after Freezy Breeze most of the time meant that you had to refresh the whole page. Instead, use a save code. After resetting to an attempt or just booting up Mike World 2, put in this specific save code, 1506060708090120141114113033; this is a fresh save code on Easy difficulty, allowing for quick resetting every time. If you are running expert, replace the first number in the save code with 4.

Next up, the hub route. Hub routes can lose or gain time, and the default 1,2,3,4,5,6 route loses a lot of time traveling to different levels. To this day, the best route in SMW2 that has short traveling times between levels is 2,3,1,4,5,6.

Finally, if you want to submit your runs onto the Speedrun.com leaderboards (https://www.speedrun.com/super_mike_world_2), please read the rules.
